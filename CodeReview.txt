#Codes-spezifischer Teil

Insgesamt wahrscheinlich sehr gute Umsetzung eines Spiels, die im Detail von unserer (nicht so fortgeschrittenen) Warte nicht sehr einfach zu beurteilen ist.

Verwendung von Modulen könnte ggf. optimiert werden: anstatt alle Module mit import pygame, time, sys auf einmal zu importieren, könnte man nur die benötigten Module importieren,
z.B. import pygame oder from pygame import sprite, event, etc.

Verwendung von Konstanten: Statt im Code Zahlenwerte zu verwenden, könnte man diese als Konstanten definieren, um die Lesbarkeit zu verbessern. 
Beispielsweise könnte man eine konstante Variable wie OBSTACLE_TIMER_INTERVAL = 1400 am Anfang des Codes definieren.

Funktional wird das Spiel ohne diese beiden Improvements natürlich aber nicht beeinträchtigt, weshalb sie ggf. auch erst bei noch intensiverer Weiterentwicklung des Spiels angedacht werden können.
